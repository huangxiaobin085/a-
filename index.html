❤️ 心法总诀

情绪为道 → 主线为法 → 抢筹为术 → 助攻为器 → 无天量不龙头

按情绪周期定仓位，按时间节点做盈亏比最高的方向，注意调整时间有节奏，技术有周期。

🎯 100分打分规则

  • 收盘红绿差值 30分：差值60→30分，40→25分，20→20分，0→15分，-20→10分，-40→5分
  • 竞价红绿 20分：绝对值12分 + 两日对比8分；红>绿且红扩额外加分
  • 连板类 25分：高度10分 + 涨停数8分 + 跌停惩罚7分
  • 涨跌家数 10分：涨家数/总股数≈5300
  • 外部事件 10分：重大事件8分，小事件5分，无事件3分
  • 额外修正 ±：人工填写额外分列

🌡️ 六阶段七周期对照

  • 六阶段：冰点 → 回暖 → 试探 → 主升 → 高潮 → 退潮
  • 七周期：婴儿(≤3板) → 成长(4-5板) → 加速(6板) → 确认(7板) → 成熟(8板+) → 混沌(龙空龙) → 衰退(退潮)

⚡ 强反转铁律（最高优先级）

  • 破7板 + 涨停≥50 → 次日强走强（偏强）
  • 跌破4板 → 次日强走弱（偏弱）
  • 高度连续3日下降 → 退潮确认

📊 差值铁律（赚亏效应周期）

  • 连续2天红差值正（竞价红>绿）→ 赚钱效应确认
  • 连续2天绿差值负（竞价绿>红）→ 亏钱效应确认
  • 两日红扩>绿扩 且 当日竞价红>绿 → 情绪回暖
  • 两日绿扩>红扩 且 当日竞价绿>红 → 情绪转弱

💪 竞价/收盘红绿强弱标准

  • 竞价红绿比 ≥ 1.2 → 强（看多积极）
  • 竞价红绿比 ≤ 0.8 → 弱（看空积极）
  • 当日绿 vs 前日绿：扩大→亏钱效应加剧；缩小→亏钱效应减弱
  • 收盘强弱与竞价是否一致，不一致为背离信号

🛡️ 止损铁律

  • 单日账户回撤超5% → 当日及次日无条件空仓
  • 连续2笔交易触发止损 → 立即空仓复盘，禁止急于翻本
  • 绝对不追高超过5%，绝不撤单改单追涨
  • 单只个股持仓周期不超过5个交易日
  • 前期高标低开5%+集体核按钮 → 至少等2天

📅 开仓停止信号（等2天）

  • 前期高标低开5%
  • 最高标集体核按钮
  • 收盘最高标大阴-13
  • 大盘大阴
  • 情绪位/缺口/联动异常
